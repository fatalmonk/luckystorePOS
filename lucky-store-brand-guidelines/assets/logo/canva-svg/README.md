## Uploading to Canva

### Method 1: SVG (Recommended for Print/Scaling)

1. Go to **Uploads** → **Upload files**
2. Drag `lucky-store-canva.svg`
3. It appears in your Uploads panel
4. Drag onto canvas
5. **Right-click → Ungroup** (if grouped)
6. You should see 4 separate layers:
   - `bag` — terracotta shopping bag shape
   - `letter-l` — white "L" path
   - `leaf` — green leaf and stem
   - `wordmark` + `tagline` — text layers

### Method 2: PNG (Easiest for Social Posts)

1. Upload any PNG from `logo/pngs/`
2. Drag onto canvas
3. Use as-is — no font worries
4. **Lock** the layer (so it doesn't move while you edit)

## Font Fix in Canva

The text layers use **Inter** and **Arial Black** fallbacks. Canva may show a default font instead.

**To fix:**
1. Click the "LUCKY STORE" text layer
2. In the top toolbar, change font to **"Montserrat Bold"** or **"Poppins Bold"** (closest free match to Inter)
3. For "FRESH TO YOUR DOOR", use **Montserrat Medium** or **Open Sans**
4. Adjust letter-spacing to `3–4` for the wordmark, `4–5` for the tagline

### Closest Canva Alternatives to Inter

| Inter Weight | Canva Alternative |
|--------------|-------------------|
| Inter Bold (700) | Montserrat Bold, Poppins Bold |
| Inter Extra Bold (800) | Montserrat Black |
| Inter Medium (500) | Montserrat Medium, Open Sans Medium |

## Using the Logo in Canva Templates

### Social Media Posts

| Canvas Size | Logo Placement | Logo Size |
|-------------|----------------|-----------|
| Instagram Feed (1080×1350) | Top-left corner | 80–100px wide |
| Instagram Story (1080×1920) | Top-right corner | 60–80px wide |
| Facebook Cover (820×312) | Top-left | 100–120px wide |
| Facebook Post (1200×1200) | Bottom-center or top-left | 100px wide |

**Tip:** Add a small white circle behind the mark-only version for visibility on photos.

### Adding a Background Circle

1. Go to **Elements** → **Shapes** → Circle
2. Color: `#faf8f5` (Ivory)
3. Size: ~110% of logo width
4. Position behind logo (Send backward)
5. Group together

## Quick Template Setup

### Instagram Feed Post

1. **Canvas:** 1080 × 1350px
2. **Background:** Upload a lifestyle photo → set as background
3. **Gradient overlay:** Add a rectangle, color `#1c1917`, opacity 60%, positioned at bottom 40%
4. **Logo:** Upload `lucky-store-mark@2x.png`, place top-left, size ~80px
5. **Headline:** "Fresh from the market" in Montserrat Bold, white, size 48px
6. **Subtitle:** "Same-day delivery in Chattogram" in Montserrat Medium, white, size 24px
7. **Price tag:** Rectangle pill, color `#dc5f3b`, text "৳45/kg · Order by 11 AM", white, bold

### Instagram Story

1. **Canvas:** 1080 × 1920px
2. **Background:** Full-bleed lifestyle photo
3. **Top sticker:** Semi-transparent dark rectangle, text "Just Arrived · Palong Shak", white
4. **Logo:** Mark-only, top-right, ~60px
5. **CTA pill:** Rounded rectangle `#dc5f3b`, text "Tap to Order", white, bold

## Common Issues & Fixes

| Problem | Cause | Fix |
|---------|-------|-----|
| Logo looks blurry | Canvas too small | Start at 1080px minimum, export at 2x |
| Font not showing | Canva doesn't have Inter | Switch to Montserrat Bold |
| Leaf looks separate | Ungrouped wrong | Regroup bag+L+leaf as one element |
| Wrong colors | Photo filter applied | Don't apply filters to the logo group |
| Text warped | Stretched canvas | Resize proportionally (lock aspect ratio) |

## Quick Export Checklist

Before downloading from Canva:

- [ ] Logo is locked in position
- [ ] Colors match brand swatches
- [ ] Font is Montserrat Bold (or similar)
- [ ] No filters applied to logo
- [ ] Canvas size matches target platform
- [ ] Export as PNG (for photos) or PDF (for print)
